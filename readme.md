## Marp

Presentation is written using marp

https://github.com/marp-team/marp-cli/

### Install

```
npm install -g @marp-team/marp-cli
```
### Start

```
npx @marp-team/marp-cli -sw .
```

### Render PDF

```
npx @marp-team/marp-cli codefreshPresentation.md --pdf --allow-local-files
```
